## Sales Data Analysis using SQL

### Tools Used
- MySQL Workbench

### Objective
To analyze sales data and extract meaningful business insights using SQL queries.

### Dataset Description
The dataset contains order-level sales information including:
- Order number and date
- Customer details
- Country
- Product line
- Sales amount

### SQL Tasks Performed
- Created database and table
- Inserted sales data
- Performed aggregation using GROUP BY
- Analyzed sales by country and product line
- Extracted year-wise sales trends
- Identified top-performing countries

### Key Insights
- Certain countries contribute higher total sales
- Classic Cars product line shows strong performance
- Sales trends vary across years
