## Sales Performance Dashboard

### Tools Used
- Microsoft Excel

### Objective
To analyze sales data and build an interactive dashboard for business decision-making.

### Analysis Performed
- Data cleaning (duplicates, blanks, formatting)
- Pivot table analysis
- Sales trends by Country, Product Line, and Year

### Key Insights
- Identified top-performing countries by sales
- Found best-selling product lines
- Analyzed yearly sales trends

### Output
- Interactive Excel dashboard with multiple charts
